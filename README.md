# Employee Leave Management System

A web-based leave management system built with Django and Django REST Framework, allowing employees to apply for leave and HR to review and approve/reject applications.

## Features
- Employee registration & login (role-based dashboards for HR and Employees)
- Apply, track, and manage leave requests
- HR dashboard with approval/rejection workflow
- Search & filter leave records
- REST APIs for Employee and Leave management

## Tech Stack
Python, Django, Django REST Framework, MySQL, HTML, CSS, Bootstrap

## Setup
``````bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
``````
